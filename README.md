# Heart Disease Prediction System

A Machine Learning web application built using Streamlit and K-Nearest Neighbors (KNN) algorithm to predict the risk of heart disease based on patient health parameters.

## Features
- Interactive Streamlit UI
- Real-time prediction
- Data preprocessing and scaling
- One-hot encoded categorical variables
- KNN classification model

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Streamlit
- Joblib

## Dataset Features
- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- ECG Results
- Max Heart Rate
- Exercise-Induced Angina
- Oldpeak
- ST Slope

## Run Locally

pip install -r requirements.txt

streamlit run app.py
